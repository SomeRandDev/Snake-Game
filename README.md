# Snake-Game

This is a simple 'Snake Game' made in C.

## Rules

1. Don't run the snake into the a wall or its tail.
2. Eat the Fruits (F) to gain lenght.


## Controlls

Up: 'W'

Down: 'S'

Left: 'A'

Right: 'D'

## How to run

Just execute the 'Game.exe' file

or

Compile the 'Game.c' file and execute it afterwards.
